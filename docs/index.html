Original CommonLibSSE

CommonLibSSE is a reverse engineered library for Skyrim Special Edition.

The library was created by Ryan McKenzie (GitHub, Nexus) in 2018.

It is released under the MIT license:

Copyright (c) 2018 Ryan-rsm-McKenzie

CommonLibSSE-NG

Maintained by Charmed Baryon (GitHub), the NG ("Next Generation") fork provided support for the SE (1.5.97), AE (1.6+), and VR editions of the game - all via a single, compiled .dll
